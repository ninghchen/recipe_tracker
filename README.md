# 🍊 Recipe Tracker — Personal Recipe Tracker

Self-project to create a place to track my own recipes built with Flask and vanilla HTML/CSS/JS.
Currently limited to running locally – not hosted with a DB.

## Features
- Multiple recipe categories
- Add ingredients with name, amount, and unit
- Optional photo uploads per recipe
- Edit and delete recipes

## Setup

### Run
```bash
python app.py
```
Then open **http://localhost:5000** in any browser.
